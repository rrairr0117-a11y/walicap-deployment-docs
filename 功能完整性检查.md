## 🔧 已修正的请求体格式

### 1. Ken Burns 效果
**修正前**：
```json
{
  "image_urls": ["图片1"],
  "audio_url": "音频"
}
```

**修正后**：
```json
{
  "images": {
    "urls": ["图片1"]
  },
  "audio": {
    "url": "音频"
  },
  "settings": {
    "magnify_factor": 0.5
  }
}
```

### 2. 图片+音频生成视频
**修正前**：
```json
{
  "image_urls": ["图片1"],
  "audio_url": "音频",
  "duration_per_image": 3
}
```

**修正后**：
```json
{
  "images": {
    "urls": ["图片1"]
  },
  "audio": {
    "url": "音频"
  },
  "settings": {
    "image_duration": 10
  }
}
```

### 3. 字幕处理
**修正前**：
```json
{
  "video_url": "视频",
  "ass": "字幕",
  "subtitle_y": 1538,
  "subtitle_height": 77
}
```

**修正后**：
```json
{
  "video_url": "视频",
  "ass": "字幕",
  "y": 1538
}
```
**说明**：`/v1/video/caption` 和 `/v1/subtitle/insert-ass` 接口不支持 `subtitle_height` 参数。
