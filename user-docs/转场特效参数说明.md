# 🎬 转场特效参数说明

## 参数概览

| 参数名 | 类型 | 默认值 | 范围 | 说明 |
|:---|:---|:---|:---|:---|
| `transition_effect` | string | `null` | 见下表 | 片段间转场类型 |
| `transition_duration` | number | `0.5` | 0.1 - 2.0 | 转场时长（秒） |

---

## 支持的转场特效

### 1️⃣ 淡入淡出类

| 特效名 | 效果描述 | 适用场景 | 示例 |
|:---|:---|:---|:---|
| `fade` | 标准淡入淡出 | ⭐ 最常用，适合所有场景 | 通用过渡 |
| `fadeblack` | 经过黑场淡入淡出 | 章节切换、严肃氛围 | 恐怖故事、纪录片 |
| `fadewhite` | 经过白场淡入淡出 | 梦境、回忆、闪回 | 回忆杀、梦境 |

### 2️⃣ 滑动类

| 特效名 | 效果描述 | 适用场景 | 示例 |
|:---|:---|:---|:---|
| `slideleft` | 从右向左滑入 | 时间推进、场景切换 | 进入新场景 |
| `slideright` | 从左向右滑入 | 回溯、反转 | 时光倒流 |
| `slideup` | 从下向上滑入 | 揭示、升华 | 揭晓答案 |
| `slidedown` | 从上向下滑入 | 压抑、下坠 | 情绪低落 |

### 3️⃣ 擦除类

| 特效名 | 效果描述 | 适用场景 | 示例 |
|:---|:---|:---|:---|
| `wipeleft` | 从左向右擦除 | 快速切换 | 快节奏剪辑 |
| `wiperight` | 从右向左擦除 | 快速切换 | 快节奏剪辑 |
| `wipeup` | 从下向上擦除 | 快速切换 | 动作场景 |
| `wipedown` | 从上向下擦除 | 快速切换 | 动作场景 |

### 4️⃣ 特殊效果类

| 特效名 | 效果描述 | 适用场景 | 示例 |
|:---|:---|:---|:---|
| `circleopen` | 圆形展开 | 聚焦、揭示 | 重点展示 |
| `circleclose` | 圆形收缩 | 结束、消失 | 故事结局 |
| `dissolve` | 溶解过渡 | 柔和、艺术感 | 纪录片、艺术片 |
| `pixelize` | 像素化过渡 | 科技感、游戏风 | 游戏视频、科技内容 |
| `radial` | 径向扫描 | 时钟、倒计时 | 时间流逝 |
| `random` | 随机选择 | 多样化 | 自动随机 |

## 使用示例

### 示例1：恐怖故事（青木原）
```json
{
    "settings": {
        "transition_effect": "fadeblack",
        "transition_duration": 0.8
    }
}
```
**效果**：黑场过渡，营造压抑恐怖氛围

### 示例2：快节奏 Vlog
```json
{
    "settings": {
        "transition_effect": "fade",
        "transition_duration": 0.3
    }
}
```
**效果**：快速淡入淡出，节奏紧凑

### 示例3：艺术纪录片
```json
{
    "settings": {
        "transition_effect": "dissolve",
        "transition_duration": 1.0
    }
}
```
**效果**：柔和溶解，文艺气息

### 示例4：游戏集锦
```json
{
    "settings": {
        "transition_effect": "pixelize",
        "transition_duration": 0.5
    }
}
```
**效果**：像素化过渡，游戏风格

### 示例5：无转场（硬切）
```json
{
    "settings": {
        "transition_effect": null
    }
}
```
**效果**：直接切换，无过渡

## 推荐配置

### 按视频类型推荐

| 视频类型 | 推荐特效 | 时长 | 说明 |
|:---|:---|:---|:---|
| 恐怖/惊悚 | `fadeblack` | 0.8-1.0s | 增强恐怖氛围 |
| Vlog/日常 | `fade` | 0.3-0.5s | 自然流畅 |
| 纪录片 | `dissolve` | 0.5-1.0s | 柔和专业 |
| 游戏视频 | `pixelize` | 0.3-0.5s | 贴合主题 |
| 教程/讲解 | `fade` 或 `null` | 0.3s 或无 | 简洁清晰 |
| 回忆/怀旧 | `fadewhite` | 0.8-1.2s | 营造回忆感 |

### 按节奏推荐

| 节奏 | 时长建议 | 特效建议 |
|:---|:---|:---|
| 快节奏 | 0.1-0.3s | `fade`, `wipe`, `slide` |
| 中等节奏 | 0.3-0.6s | `fade`, `dissolve` |
| 慢节奏 | 0.6-1.2s | `fadeblack`, `fadewhite`, `dissolve` |
| 极慢/艺术 | 1.0-2.0s | `dissolve`, `fade` |

---

## 注意事项

1. **转场时长不宜过长**：建议不超过1秒，否则会影响节奏
2. **与片段时长匹配**：转场时长不应超过片段时长的1/3
3. **风格统一**：同一视频建议使用同一种转场特效
4. **null 即硬切**：不传参数或传 `null` 则无转场
