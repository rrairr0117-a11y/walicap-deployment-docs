## 🚀 安装步骤

### 步骤 1: 下载镜像文件

从上述云盘下载对应版本的 `.tar` 文件到本地。

**推荐**：下载 GPU 版本 `walicap-v2.1cuda-docker-image.tar`（性能更好）

### 步骤 2: 加载镜像到 Docker

打开终端/命令行，进入镜像文件所在目录，执行：

#### Windows (PowerShell)
```powershell
# 加载 GPU 版本
docker load -i walicap-v2.1cuda-docker-image.tar

# 或加载 CPU 版本
docker load -i walicap-v2.0-docker-image.tar
```

#### Linux / macOS
```bash
# 加载 GPU 版本
docker load -i walicap-v2.1cuda-docker-image.tar

# 或加载 CPU 版本
docker load -i walicap-v2.0-docker-image.tar
```

**预计时间**：根据文件大小，需要 2-5 分钟

### 步骤 3: 验证镜像加载成功

```bash
docker images | grep walicap
```

**预期输出**：
```
crpi-gpb2zeuxu5n5h1lc.cn-chengdu.personal.cr.aliyuncs.com/rrairr/walicap   v2.1cuda   65280eccc9de   28.7GB
```

### 步骤 4: 下载部署配置文件

从 GitHub 下载部署文件：

```bash
git clone https://github.com/rrairr0117-a11y/walicap-deployment-docs.git
cd walicap-deployment-docs
```

**或者**手动下载：
1. 访问 https://github.com/rrairr0117-a11y/walicap-deployment-docs
2. 点击 `Code` → `Download ZIP`
3. 解压到本地目录

### 步骤 5: 配置环境变量

```bash
# 复制环境变量示例文件
cp .env.nca.local.example .env.nca.local

# 编辑配置文件，填入授权信息
# Windows: notepad .env.nca.local
# Linux/Mac: nano .env.nca.local
```

**必填配置项**：
```bash
API_KEY=your_api_key_here
LICENSE_API_SECRET=sk-your_license_api_secret
LICENSE_PRODUCT_ID=65
LICENSE_KEY=LIC_your_license_key
LICENSE_USERNAME=your_username
```

> 💡 授权信息请从 https://rrairr.cn 购买后获取

### 步骤 6: 启动服务

```bash
docker-compose up -d
```

### 步骤 7: 验证服务运行

```bash
# 查看容器状态
docker-compose ps

# 查看日志
docker-compose logs -f ncat

# 测试 API
curl http://localhost:8080/v1/toolkit/test
```

**预期响应**：
```json
{
  "status": "success",
  "message": "NCA Toolkit API is running"
}
```

## 🔧 常见问题

### Q1: 加载镜像时提示 "no space left on device"

**原因**：磁盘空间不足

**解决方案**：
```bash
# 清理未使用的 Docker 资源
docker system prune -a

# 检查磁盘空间
df -h
```

### Q2: 加载镜像时提示 "invalid tar header"

**原因**：下载文件损坏或不完整

**解决方案**：
1. 重新下载镜像文件
2. 验证文件完整性（见下方 MD5 校验）

### Q3: GPU 版本无法使用 GPU

**原因**：未安装 nvidia-docker

**解决方案**：
```bash
# 安装 nvidia-docker（Linux）
distribution=$(. /etc/os-release;echo $ID$VERSION_ID)
curl -s -L https://nvidia.github.io/nvidia-docker/gpgkey | sudo apt-key add -
curl -s -L https://nvidia.github.io/nvidia-docker/$distribution/nvidia-docker.list | sudo tee /etc/apt/sources.list.d/nvidia-docker.list
sudo apt-get update && sudo apt-get install -y nvidia-docker2
sudo systemctl restart docker

# 验证 GPU 可用
docker run --rm --gpus all nvidia/cuda:11.0-base nvidia-smi
```

### Q4: 容器启动失败

**检查步骤**：
1. 确认 `.env.nca.local` 文件已正确配置
2. 检查端口 8080、9000、9001 是否被占用
3. 查看详细日志：`docker-compose logs ncat`
