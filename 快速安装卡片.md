# 🚀 瓦力聚合剪辑系统 - 快速安装卡片

> 打印或保存此卡片，方便快速部署

---

## 📦 方式一：在线安装（推荐）

### 1️⃣ 拉取镜像
```bash
# GPU 版本（推荐）
docker pull crpi-gpb2zeuxu5n5h1lc.cn-chengdu.personal.cr.aliyuncs.com/rrairr/walicap:v2.1cuda

# CPU 版本
docker pull crpi-gpb2zeuxu5n5h1lc.cn-chengdu.personal.cr.aliyuncs.com/rrairr/walicap:v2.0
```

### 2️⃣ 下载部署文件
```bash
git clone https://github.com/rrairr0117-a11y/walicap-deployment-docs.git
cd walicap-deployment-docs
```

### 3️⃣ 配置环境
```bash
cp .env.nca.local.example .env.nca.local
# 编辑 .env.nca.local，填入授权信息
```

### 4️⃣ 启动服务
```bash
docker-compose up -d
```

### 5️⃣ 测试
```bash
curl http://localhost:8080/v1/toolkit/test
```

## 💾 方式二：离线安装

### 1️⃣ 下载镜像文件
- 百度网盘：`https://pan.baidu.com/s/xxxxx`
- 阿里云盘：`https://www.aliyundrive.com/s/xxxxx`

### 2️⃣ 加载镜像
```bash
docker load -i walicap-v2.1cuda-docker-image.tar
```

### 3️⃣ 下载部署文件
从 GitHub 下载或手动下载 ZIP：
```
https://github.com/rrairr0117-a11y/walicap-deployment-docs
```

### 4️⃣ 配置并启动
```bash
cp .env.nca.local.example .env.nca.local
# 编辑配置文件
docker-compose up -d
```

---

## 🔑 必填配置项

编辑 `.env.nca.local` 文件：

```bash
API_KEY=your_api_key_here
LICENSE_API_SECRET=sk-your_license_api_secret
LICENSE_PRODUCT_ID=65
LICENSE_KEY=LIC_your_license_key
LICENSE_USERNAME=your_username
```

> 💡 授权信息从 https://rrairr.cn 获取
